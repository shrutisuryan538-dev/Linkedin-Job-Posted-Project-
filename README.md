# Linkedin-Job-Posted-Project-

## Global Hiring Trend & Salary Insights Dashboard

## Project Overview
This project presents an interactive dashboard to analyze global hiring trends and salary insights. It helps understand job distribution, company hiring patterns, and compensation trends using clear and impactful visualizations.

## Key Features
📈 Track total jobs, companies, and applications
🌍 Analyze remote vs onsite job distribution
🏢 Identify top hiring companies
📊 Explore company size distribution
🎯 Understand jobs by experience level
💰 View average, median salary & salary spread
🌎 Compare salary across countries
📉 Analyze salary by experience level & company size
🔍 Interactive filters for better insights

## Key Insights
-Mid-Senior level roles have the highest demand
-Remote jobs account for a smaller share compared to onsite roles
-Canada offers the highest salary range among selected countries
-Entry-level roles have significantly lower salaries compared to experienced roles
